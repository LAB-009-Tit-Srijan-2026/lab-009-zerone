
  # Design LearnAI LMS UI

  This is a code bundle for Design LearnAI LMS UI. The original project is available at https://www.figma.com/design/HyWPoVJ4ylcK0ZnagLm3ss/Design-LearnAI-LMS-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## OpenAI API Key

  Copy `.env.example` to `.env` and set your OpenAI key:

  ```env
  VITE_OPENAI_API_KEY=your_real_openai_api_key
  ```
