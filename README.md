# 🎥 Real-time Q&A Video Chat Platform

A production-ready, AI-powered educational SaaS platform designed for students, educators, and online learning environments to enable contextual video learning, real-time AI-assisted question answering, smart note-taking, lecture bookmarking, and interactive educational experiences.

## 🎯 Key Features

### Core Learning Dashboards

1. **Dashboard** – Comprehensive student learning overview with:
   - Learning progress metrics
   - Recently watched lectures
   - Course completion insights
   - Productivity tracking
   - Quick access navigation
   - Activity monitoring

2. **Course Management** – Organized educational workflow featuring:
   - Course cards with thumbnails
   - Progress indicators
   - Lecture categorization
   - Smart navigation system
   - Search & filtering support

3. **Lecture Experience** – Interactive learning environment featuring:
   - Embedded video player
   - AI-powered contextual chat
   - Real-time lecture interaction
   - Lecture metadata display
   - Responsive educational layout

4. **AI Learning Assistant** – Intelligent educational support system:
   - Answers contextual questions instantly
   - Explains lecture concepts in real time
   - Assists self-paced learning
   - Provides conversational tutoring support
   - Improves understanding during lectures

5. **Notes Management** – Smart educational productivity tool:
   - Create lecture notes
   - Organize notes by course
   - Edit & manage learning content
   - Add timestamps and tags
   - Improve revision workflow

6. **Bookmark System** – Timestamp-based lecture navigation:
   - Save important lecture moments
   - Quick timestamp navigation
   - Highlight important concepts
   - Faster revision experience

7. **Learning History** – Educational activity tracking:
   - Recently viewed lectures
   - Learning continuation support
   - Session history tracking
   - Progress monitoring

8. **Settings & Preferences** – Personalized learning experience:
   - Profile customization
   - Notification settings
   - Playback controls
   - User experience personalization
   - Learning preference management

## 🚀 Technology Stack

- **Framework**: React with TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **UI Components**: Radix UI primitives
- **Charts & Visualizations**: Recharts
- **Icons**: Lucide React
- **Forms**: React Hook Form
- **Animations**: Framer Motion
- **Notifications**: Sonner
- **Utilities**: Date-fns, React DnD

## ⚙️ Setup & Installation

Follow these steps to run the project locally for development or evaluation.

### Prerequisites

Make sure the following are installed:

- Node.js (v18 or higher recommended)
- npm or yarn
- Modern browser

Check installation:

```bash
node -v
npm -v
```

---

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/real-time-qa-video-chat.git
cd real-time-qa-video-chat
```

---

### 2. Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

### 3. Run Development Server

```bash
npm run dev
```

The app will run on:

```bash
http://localhost:5173
```

---

### 4. Production Build

Create optimized production build:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```
